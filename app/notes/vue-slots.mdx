export const title = Vue slots

Vue components can expose values to their children with `v-bind`

```xml
<template>
  <ul>
    <slot v-bind="{ items }" />
  </ul>
</template>
```

```xml
<ListComponent v-slot="{ items }">
  <li v-for="item in items"> { item } </li>
</ListComponent>
```

It's important to note that you can only bind an object. No arrays, and no primitives.

```xml
<template>
  <ul>
    <slot v-bind="items" /> // BAD
  </ul>
</template>
```
